A set of tools that can perform analysis of market structure, and build models that programmatically detect opportune trade setups.

1. Detects price swings based on candle bodies, including volume imbalances and renders them in blue.
2. Inside these swings, detects FVGs that include volume imbalances and renders them in greed or red.
3. Calculates OTE levels for every swing and renders them in yellow.
